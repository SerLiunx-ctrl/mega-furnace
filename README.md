# Mega Furnace

A Factorio 2.0 mod focused on large-scale late-game smelting.

## Notes

- Recommended for mid-to-late game factory builds.
- Best used when you want to reduce the footprint of large furnace arrays.
- Current balance is tuned around high throughput with a clear upgrade path.
- Supports optional `Space Age` logistics integration when the DLC is enabled.
- Supports optional `AAI Loaders` integration with a matching high-throughput loader tier.

## Installation

1. Put the mod folder or zip file into your Factorio `mods` directory.
2. Make sure `info.json` is at the root of the mod package.
3. Enable the mod in-game.

## Compatibility

- Tested with `Factorio 2.0.76`
- Requires `base >= 2.0.0`
- Optional support for `space-age >= 2.0.0`
  Uses Space Age logistics and processor content when available.
- Optional support for `aai-loaders >= 0.2.0`
  Adds a compatible high-throughput loader and hooks it into the high-throughput logistics tier.
